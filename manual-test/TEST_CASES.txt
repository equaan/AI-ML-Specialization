MediAgent Manual Test Cases

Purpose:
Use these cases to validate the end-to-end app behavior with real assets from this repository.

Where to run:
1) Open the app UI.
2) Go to New Analysis.
3) For each case, use the exact suggested input.
4) After run, verify Results, Evidence, Lab Report, and Records pages.

Important:
- data/raw/mimic_cxr is empty in this workspace, so no chest X-ray image is included yet.
- The image cases below use HAM10000 skin lesion images currently available.

POST-RUN CHECKLIST (Apply after any case)
==================================================
1) Refresh browser page.
   Expected: Latest report still available (persistent history loaded).

   Expected: Selected session loads back into current report view.

   Expected: JSON, Markdown, and PDF export endpoints respond for that session.

4) Check backend health endpoint.
   Expected: /api/health returns valid JSON response (ok or degraded, but not server error).
